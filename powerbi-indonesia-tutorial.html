Instalasi & Setup Power BI Desktop

1

Download Power BI Desktop

Buka Microsoft Store di Windows 10/11 dan cari "Power BI Desktop" — gratis, tidak perlu lisensi untuk membuat dan mengedit. Atau download langsung dari powerbi.microsoft.com/desktop. Pilih versi terbaru (minimal versi Desember 2024 untuk mendukung fitur Copilot dan visual terbaru).

ℹ
Power BI Desktop berjalan di Windows saja. Pengguna Mac harus memakai Windows VM atau menggunakan Power BI Service (web) dengan fitur terbatas.
2

Login dengan Akun Microsoft

Saat pertama buka, Power BI meminta login. Gunakan akun Microsoft gratis (@outlook.com atau @hotmail.com). Untuk fitur publish, butuh email kerja atau sekolah — akun personal tidak bisa publish ke Power BI Service.

⚠
Untuk keperluan portofolio atau berbagi publik, buat akun Microsoft 365 Developer (gratis 90 hari) di developer.microsoft.com/microsoft-365/dev-program.
3

03
ETL

Transformasi Data di Power Query Editor

Data dari BPS sering punya header ganda, baris kosong, tipe data salah, dan format tidak konsisten. Power Query adalah tempat kita membersihkan semua itu sebelum dianalisis.

1

Bersihkan Header Ganda (Masalah Umum Data BPS)

File Excel BPS biasanya punya 2–3 baris header. Di Power Query: klik Home → Remove Top Rows untuk hapus baris metadata. Lalu Home → Use First Row as Headers untuk jadikan baris pertama sebagai nama kolom.

M Language — Power Query
Hapus Header BPS
// Hapus 3 baris teratas (biasanya berisi judul tabel BPS)
= Table.Skip(Source, 3),

// Jadikan baris pertama sebagai header kolom
= Table.PromoteHeaders(#"Removed Top Rows", 
  [PromoteAllScalars=true]),

// Hapus baris kosong di bawah
= Table.SelectRows(#"Promoted Headers", 
  each [Tahun] <> null and [Tahun] <> "")
ℹ
Di UI Power Query (tanpa perlu tulis M): klik kanan header kolom → Remove untuk hapus kolom, atau gunakan tombol di ribbon Home tab untuk operasi umum.
2

Perbaiki Tipe Data Kolom

Power BI sering salah mendeteksi tipe data angka BPS sebagai Text karena ada karakter spesial. Klik kanan header kolom → Change Type:

KolomTipe DataCatatan
TahunWhole NumberBukan Text!
Bulan / KuartalText atau Whole NumberTergantung format sumber
Growth_YoY, Inflasi_%Decimal NumberGanti koma dengan titik jika perlu
Nilai_Ekspor_USDDecimal NumberHapus karakter "$" dan "," dulu
Kode_ProvinsiTextJangan ubah ke angka (ada kode "01" dst)
Nama_ProvinsiTextStandarisasi penulisan nama
3

Buat Kolom Tanggal (Date Key)

Kita perlu kolom tanggal untuk relasi ke tabel kalender. Di tbl_PDB, tambahkan kolom baru: Add Column → Custom Column:

M Language
Buat DateKey dari Tahun + Kuartal
// Nama kolom baru: TanggalAwal
// Ubah Q1=Jan, Q2=Apr, Q3=Jul, Q4=Oct
= if [Kuartal] = "Q1" then 
    Date.From(Text.From([Tahun]) & "-01-01")
  else if [Kuartal] = "Q2" then
    Date.From(Text.From([Tahun]) & "-04-01")
  else if [Kuartal] = "Q3" then
    Date.From(Text.From([Tahun]) & "-07-01")
  else
    Date.From(Text.From([Tahun]) & "-10-01")
4

Standarisasi Nama Provinsi

Nama provinsi di BPS tidak selalu konsisten antar dataset. Gunakan Transform → Replace Values untuk menyamakan:

M Language
Standardisasi Nama Provinsi
// Ganti variasi penulisan agar konsisten
= Table.ReplaceValue(
    Table.ReplaceValue(
      Table.ReplaceValue(prev,
        "DKI Jakarta", "Jakarta", 
        Replacer.ReplaceText, {"Nama_Provinsi"}),
      "Kep. Bangka Belitung", "Bangka Belitung",
      Replacer.ReplaceText, {"Nama_Provinsi"}),
    "D.I Yogyakarta", "DI Yogyakarta",
    Replacer.ReplaceText, {"Nama_Provinsi"})
5

Buat dim_Waktu (Tabel Kalender)

Di Power Query: Home → New Source → Blank Query. Beri nama dim_Waktu dan masukkan kode berikut di Advanced Editor:

M Language
dim_Waktu — Tabel Kalender Lengkap
let
  StartDate = #date(2015, 1, 1),
  EndDate   = #date(2026, 12, 31),
  
  DayCount  = Duration.Days(EndDate - StartDate) + 1,
  DateList  = List.Dates(StartDate, DayCount, 
                #duration(1,0,0,0)),
  
  DateTable = Table.FromList(DateList, 
                Splitter.SplitByNothing()),
  
  // Rename kolom pertama
  Renamed = Table.RenameColumns(DateTable, {{"Column1", "Tanggal"}}),
  
  // Ubah ke tipe Date
  TypeDate = Table.TransformColumnTypes(Renamed, 
               {{"Tanggal", type date}}),
  
  // Tambahkan kolom-kolom berguna
  AddTahun = Table.AddColumn(TypeDate, "Tahun", 
               each Date.Year([Tanggal]), Int64.Type),
  AddBulan = Table.AddColumn(AddTahun, "Bulan", 
               each Date.Month([Tanggal]), Int64.Type),
  AddNamaBulan = Table.AddColumn(AddBulan, "NamaBulan", 
               each Date.MonthName([Tanggal"], "id-ID"), type text),
  AddKuartal = Table.AddColumn(AddNamaBulan, "Kuartal", 
               each "Q" & Text.From(Date.QuarterOfYear([Tanggal])), 
               type text),
  AddSemester = Table.AddColumn(AddKuartal, "Semester",
               each if Date.Month([Tanggal]) <= 6 
                    then "S1" else "S2", type text)
in
  AddSemester

Setelah selesai klik Close & Apply di ribbon untuk memuat semua query ke data model.

05
DAX Formula

Membuat Measures DAX — Kalkulasi Utama

DAX (Data Analysis Expressions) adalah bahasa formula Power BI. Buat semua measure di folder [_Measures] agar mudah dikelola. Klik kanan tabel → New Measure.

1

Measures Dasar — PDB

DAX
Measures Pertumbuhan PDB
// Pertumbuhan PDB terkini (nilai terakhir)
[PDB Growth Terkini %] =
LASTNONBLANK(
  tbl_PDB[Growth_YoY], 
  CALCULATE(MAX(tbl_PDB[Growth_YoY]))
)

// Rata-rata growth sepanjang tahun yang dipilih
[PDB Growth Rata2 %] = 
AVERAGE(tbl_PDB[Growth_YoY])

// PDB terbaru dalam Triliun Rupiah
[PDB ADHB Terkini] =
CALCULATE(
  MAX(tbl_PDB[PDB_ADHB_T]),
  LASTDATE(dim_Waktu[Tanggal])
)

// YoY change untuk sparkline (bandingkan tahun ini vs tahun lalu)
[PDB Growth YoY Change] =
VAR CurrentYear = SELECTEDVALUE(dim_Waktu[Tahun])
VAR CurrentGrowth = CALCULATE(
  AVERAGE(tbl_PDB[Growth_YoY]),
  dim_Waktu[Tahun] = CurrentYear)
VAR PrevGrowth = CALCULATE(
  AVERAGE(tbl_PDB[Growth_YoY]),
  dim_Waktu[Tahun] = CurrentYear - 1)
RETURN
  CurrentGrowth - PrevGrowth
2

Measures Inflasi

DAX
Measures Inflasi
// Inflasi YoY terkini
[Inflasi YoY Terkini %] =
CALCULATE(
  MAX(tbl_Inflasi[Inflasi_YoY]),
  tbl_Inflasi[Kelompok] = "Umum",
  LASTDATE(dim_Waktu[Tanggal])
)

// Inflasi MoM (month-over-month)
[Inflasi MoM %] =
CALCULATE(
  MAX(tbl_Inflasi[Inflasi_MoM]),
  tbl_Inflasi[Kelompok] = "Umum",
  LASTDATE(dim_Waktu[Tanggal])
)

// Status inflasi (untuk conditional formatting)
[Status Inflasi] =
VAR Inf = [Inflasi YoY Terkini %]
RETURN
  IF(Inf <= 3, "Terkendali",
    IF(Inf <= 5, "Sedang", "Tinggi"))
3

Measures Ekspor-Impor

DAX
Measures Neraca Perdagangan
// Total ekspor periode terpilih
[Total Ekspor USD] = SUM(tbl_EksporImpor[Ekspor_USD])

// Total impor
[Total Impor USD] = SUM(tbl_EksporImpor[Impor_USD])

// Neraca perdagangan
[Neraca Dagang USD] = 
  [Total Ekspor USD] - [Total Impor USD]

// Status surplus/defisit
[Status Neraca] =
  IF([Neraca Dagang USD] > 0, "✅ Surplus", "🔴 Defisit")

// Growth ekspor YoY
[Ekspor Growth YoY %] =
VAR CurrEkspor = [Total Ekspor USD]
VAR PrevEkspor = CALCULATE(
  [Total Ekspor USD],
  SAMEPERIODLASTYEAR(dim_Waktu[Tanggal]))
RETURN
  DIVIDE(CurrEkspor - PrevEkspor, PrevEkspor) * 100
4

Measures Sosial (IPM, Kemiskinan, TPT)

DAX
Measures Indikator Sosial
// IPM Nasional terkini
[IPM Nasional] =
CALCULATE(
  AVERAGE(tbl_IPM[IPM_Score]),
  tbl_IPM[Provinsi] = "Indonesia",
  LASTDATE(dim_Waktu[Tanggal])
)

// Provinsi dengan IPM tertinggi
[Provinsi IPM Tertinggi] =
MAXX(
  TOPN(1, 
    SUMMARIZE(tbl_IPM, tbl_IPM[Provinsi], 
               "MaxIPM", MAX(tbl_IPM[IPM_Score])),
    [MaxIPM], DESC),
  tbl_IPM[Provinsi]
)

// TPT Nasional
[TPT Nasional %] =
CALCULATE(
  MAX(tbl_Ketenagakerjaan[TPT_Persen]),
  tbl_Ketenagakerjaan[Provinsi] = "Indonesia",
  LASTDATE(dim_Waktu[Tanggal])
)

// Persentase kemiskinan nasional
[Kemiskinan Nasional %] =
CALCULATE(
  MAX(tbl_Kemiskinan[Miskin_Persen]),
  tbl_Kemiskinan[Provinsi] = "Indonesia",
  LASTDATE(dim_Waktu[Tanggal])
)

// Perubahan kemiskinan YoY (dalam pp)
[Δ Kemiskinan pp] =
VAR Now  = [Kemiskinan Nasional %]
VAR Prev = CALCULATE(
  MAX(tbl_Kemiskinan[Miskin_Persen]),
  tbl_Kemiskinan[Provinsi] = "Indonesia",
  SAMEPERIODLASTYEAR(dim_Waktu[Tanggal]))
RETURN Now - Prev

Waterfall Chart — Kontribusi per Sektor

Layout Canvas & Grid

4

Conditional Formatting — Warna Otomatis

Pada setiap KPI Card dan tabel: klik visual → Format → klik fx di sebelah warna data → pilih "Field value" atau "Rules":

DAX
Measure untuk Warna Kondisional
// Warna untuk KPI Card PDB Growth
[Warna PDB] =
VAR G = [PDB Growth Terkini %]
RETURN
  SWITCH(TRUE(),
    G >= 5,   "#27AE60",  // hijau: di atas target
    G >= 4,   "#F39C12",  // oranye: mendekati target

UX

Interaktivitas, Slicer & Filter

1

Slicer Tahun (Dropdown)

Visual: Slicer. Field: dim_Waktu[Tahun]. Format → Style: Dropdown. Slicer ini akan memfilter semua visual di halaman secara otomatis berkat relasi yang sudah kita buat.

ℹ
Untuk membuat slicer berlaku di semua halaman: klik kanan slicer → Sync Slicers → aktifkan di semua halaman yang diinginkan. Ini menghindari pengguna harus reset filter setiap ganti halaman.
2

Tombol Navigasi Antar Halaman

Buat tombol navigasi di header setiap halaman: Insert → Buttons → Navigator → Page Navigator. Power BI otomatis membuat tombol untuk setiap halaman. Atau buat manual: Insert → Button → pilih ikon → Action → Type: Page navigation → Destination: pilih halaman.

3

Drill-through — Klik Provinsi untuk Detail

Buat halaman baru bernama "Detail Provinsi" (sembunyikan dari nav). Di halaman tersebut, tambahkan field Nama_Provinsi ke bagian Drill-through filters. Kini pengguna bisa klik kanan provinsi di map → Drill through → Detail Provinsi.

4

Tooltip Custom — Hover untuk Detail

Buat halaman baru, set Page type: Tooltip. Isi dengan visual kecil (mini line chart, detail angka). Di visual utama (misalnya map): Format → Tooltips → Type: Report page → pilih halaman tooltip. Kini hover di atas provinsi menampilkan popup visual lengkap.

✓
Tooltip halaman adalah fitur yang paling membuat dashboard terkesan profesional. Satu hover bisa menampilkan tren 5 tahun, bukan sekedar satu angka.
5

Bookmarks — Tampilan Preset

View → Bookmarks. Buat bookmark untuk kondisi filter berbeda, contoh: "2025 — Q1", "2025 — Full Year", "Fokus Jawa", "Fokus Luar Jawa". Hubungkan ke tombol di canvas menggunakan Button → Action → Bookmark. Pengguna tinggal klik tombol untuk beralih perspektif.

Pro Tips

Tips & Best Practices Power BI

#TipsKenapa Penting
01 Selalu gunakan Measures, bukan calculated columns untuk agregasi Measures dihitung saat render (cepat), calculated columns dihitung saat load (lambat, makan RAM)
02 Matikan "Auto date/time" di Options File → Options → Data Load → matikan. Ini mencegah Power BI membuat tabel tanggal otomatis yang memakan memori ganda
03 Gunakan DIVIDE() bukan operator / DIVIDE(a, b) otomatis handle error division by zero. Tanpa ini dashboard bisa error saat data kosong
04 Format angka besar dengan satuan Di Format → Display units: pilih "Trillions" atau "Billions". Atau custom: "Rp #,##0.0 T" agar terbaca "Rp 22.1 T" bukan "22139000000000"
05 Beri Alt Text pada setiap visual Format → General → Alt text. Wajib untuk aksesibilitas dan jika dashboard dibagikan ke pejabat/publik
06 Gunakan Performance Analyzer View → Performance Analyzer → Start Recording → refresh. Temukan visual paling lambat dan optimasi DAX-nya
07 Jangan lebih dari 8 visual per halaman Dashboard padat sulit dibaca. Bagi ke sub-halaman. Setiap halaman harus punya 1 pertanyaan utama yang dijawab
08 Simpan PBIX secara berkala dengan Ctrl+S Power BI tidak punya auto-save. Simpan setiap 15 menit atau setelah setiap langkah penting
⭐
DAX Studio: Install tool gratis daxstudio.org untuk debug DAX, cek query plan, dan ukur performa. Ini tool wajib untuk Power BI developer serius.

Deploy

Publish & Bagikan Dashboard

1

Publish ke Power BI Service

Di Power BI Desktop: Home → Publish. Pilih workspace (gunakan "My Workspace" untuk personal). File .pbix akan diupload ke cloud Power BI. Akses di app.powerbi.com dari browser manapun.

2

Buat Dashboard dari Report

Di Power BI Service, buka report → hover di visual → klik ikon pin 📌 → pilih "Existing Dashboard" atau "New Dashboard". Dashboard adalah kumpulan tiles dari berbagai report — bagus untuk tampilan overview satu layar (TV/monitor besar).

3

Embed ke Website / SharePoint

Di Power BI Service: klik report → File → Embed report → Website or portal. Copy kode iframe dan tempel ke website/intranet Anda. Untuk akses publik tanpa login: gunakan Publish to web (tapi pastikan data tidak sensitif).

Setup Auto-Refresh Data

Agar data selalu update otomatis tanpa harus upload ulang: di Power BI Service → Dataset → Settings → Scheduled Refresh. Untuk file XLSX lokal, Anda perlu install Power BI Gateway di komputer yang terhubung ke file. Set refresh harian atau mingguan sesuai frekuensi update BPS.
